# Como Funciona a Internet

## Redes

**backbone** => Espinha dorsal que gerencia tráfego da internet
**provedor de acesso** => permite acesso ao backbone
**IP** 127.0.0.1 => local host

### TCP/IP
- TCP => Protocolo controle de transmissão 
- IP => Protocolo da internet
- 4 camadas =>  1.física (placa de rede/cabo de rede)
		2.rede (ip)
		3.transporte (TCP/UDP)
		4.aplicação (FTP/SMTP/HTTP)

- UDP => rápido, não confiável, carro ovo - desordenado/não se sabe quem recebe a mensagem, envio desordenado dos pacotes, livestream

- TCP => voltado a conexão, handshake, integridade, ordem dos dados, aplicativo de mensagem de texto


### Portas

- "portas" entrada e saída de dados

- 20: FTP
- 22: SSH
- 25: SMTP
- 53: DNS
- 80: HTTP
- 443: HTTPS
- PORTA MYSQL: 3306
- APACHE: 80
- GMAIL: 25, 465, 587
